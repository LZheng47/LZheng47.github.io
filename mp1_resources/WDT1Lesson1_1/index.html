Web Technologies

HTML

HTML is Hypertext Markup Language, and it is the standard markup language used to create webpages. HTML designs the structure of the webpage, laying out where to display elements of a webpage, including paragraphs, subtitles, headings, pictures, and etc. This language also formats the general way of how texts are displayed, such as if they are bolded, the spacing between the texts, and etc.

CSS

CSS is Cascading Style Sheets, a language that helps modify the style of HTML elements. They can be used to specify colors and fonts of text. CSS files can be reused, which means they guarantee efficiency, allowing for different HTML webpages to use the same style.

JavaScript

JavaScript, or JS, is a programming language that makes webpages interactive, updating the content that are on the webpages. It allows for certain events to happen when an action is done, such as when the user clicks, code in JavaScript can make it so that user can type something and it updates a text. On webpages, JavaScript can be used with HTML and CSS, allowing for animations, controlling multimedia, and etc.

jQuery

jQuery is a JavaScript library and also an open-source software, implemented by referencing it in the head section of the HTML file. jQuery is compatable with many different web browsers, making it easier for web designers to code rather than modifying it specifically for each. jQuery lets information on webpage update without the need to refresh or move different elements on a page.

Bootstrap

Bootstrapping describes a process that automatically executes a series of commands. An example of this is the startup to a computer, since when it is turned on, the computer automatically runs commands that powers on the hardware, loads the UEFI firmware, and other processes are initialized. Bootstrap can also be used to decribe applications when they are opened, iniating commands that may check for updates or load certain libraries.
